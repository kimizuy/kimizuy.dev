## はじめに

ほとんどの VSCode 利用者がデフォルトフォーマッターに Prettier を指定しています（たぶん）。

そんな場合に VSCode で schema.prisma といったファイルのフォーマットを有効にする方法を紹介します。

## 方法

- まずは [Prisma の拡張機能](https://marketplace.visualstudio.com/items?itemName=Prisma.prisma) を VSCode にインストールします。
- 次に VSCode の設定ファイルに以下の行を追加します。

```json:settings.json
  "[prisma]": { "editor.defaultFormatter": "Prisma.prisma"},
```

以上！
